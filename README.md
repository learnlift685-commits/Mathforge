# MATHFORGE NRW V0.4 — UX REBUILD

V0.4 ordnet die vollständige Lernlogik aus V0.3 neu, ohne Funktionen zu entfernen.

## Hauptnavigation

- Heute
- Lernen
- Üben
- Klausuren
- Fortschritt
- Einstellungen

## Grundprinzip

Die App zeigt zuerst nur die nächste sinnvolle Handlung. Tiefe Werkzeuge werden erst im passenden Kontext eingeblendet. Dadurch bleibt MATHFORGE umfangreich, wirkt aber nicht mehr wie ein Cockpit mit zu vielen gleichwertigen Schaltern.

## Enthaltene Systeme

Alle Systeme aus V0.3 bleiben erhalten: Masterclasses, Generatoren, Rechenweg-Rubrik, Teilpunkte, Warum-Labore, Diagnose, Graphen-Labor, Kurvendiskussions-Simulator, 3D-Vektorraum, Klausurzentrum, Fehlerlabor, Formelwerk, adaptive Wiederholung und Wochenplan.

## GitHub Pages

Alle Dateien dieser Version liegen flach nebeneinander und können direkt in die Repository-Wurzel hochgeladen werden. Es ist kein Build-Schritt erforderlich.

## Speicherung

Der Lernstand bleibt lokal im Browser. V0.3-Daten werden automatisch übernommen.
