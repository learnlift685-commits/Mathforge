# MATHFORGE NRW V0.1 – GitHub Flat

## Direkt veröffentlichen

1. ZIP entpacken.
2. Alle Dateien **nebeneinander** in die oberste Ebene des GitHub-Repositories hochladen.
3. GitHub: Settings → Pages → Deploy from a branch → main → /(root).
4. Speichern und die erzeugte Pages-Adresse öffnen.

Es gibt keinen src-, dist-, assets- oder node_modules-Ordner und keinen Build-Befehl.

## Enthalten

- vollständiger Atlas der 31 verbindlichen NRW-EF-Kompetenzen (KLP Mathematik 2023)
- 21 umfangreiche aktive Lernmodule
- Theorie, Formeln, Worked Examples, typische Fehler und interaktive Aufgaben
- Eingangsdiagnose
- adaptive Smart-Practice-Mischung
- Mastery, XP, Fehler-Tresor und zeitversetzte Wiederholung
- Graphen-Labor für Polynom, Parabel und Sinus samt Ableitung
- durchsuchbares Formelwerk
- lokale Speicherung ohne Anmeldung oder Cloud
- Dark/Light Design und iPad-Navigation

## Technischer Hinweis

Mathematische Formeln werden durch MathJax aus einem CDN gerendert. Die App selbst und der Lernstand funktionieren ohne Backend; für die erstmalige Formelbibliothek ist eine Internetverbindung nötig. Falls das CDN blockiert ist, bleiben die LaTeX-Ausdrücke lesbar, werden aber nicht typografisch gesetzt.
