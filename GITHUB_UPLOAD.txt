MATHFORGE NRW V0.3 AUF GITHUB PAGES
====================================

1. ZIP-Datei entpacken.
2. Alle enthaltenen Dateien direkt in die oberste Ebene deines GitHub-Repositorys hochladen.
3. In GitHub zu Settings -> Pages wechseln.
4. Deploy from a branch auswählen.
5. Branch main und / (root) auswählen.
6. Speichern und kurz warten.

WICHTIG
- Keine Datei umbenennen.
- Keine der JavaScript-Dateien weglassen.
- Es ist kein npm install und kein Build-Befehl nötig.
- Ein vorhandener V0.2-Lernstand wird automatisch übernommen.
